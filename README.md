# GeoMathic
GeoMathic is for drawing functional graph like GeoGebra

# Install
```python
pip install tkinter
pip install sympy
pip install matplotlib
pip install numpy
```
# Quick Start
- download and run main.py
- enter function value (e.g. y = 2x +3)
- enter range as two integer
- plot graphs
